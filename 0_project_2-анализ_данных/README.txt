Итоговый проект по юниту Разведывательный анализ данных. Суть проекта — отследить влияние условий жизни учащихся в возрасте от 15 до 22 лет на их успеваемость по математике, чтобы на ранней стадии выявлять студентов, находящихся в группе риска.

У нас имеется датасет с набором учеников и их параметрами. Проводим первичную обработку этих данных, корреляционный анализ и анализ номинативных переменных. На основании проделанной работы делаем вывод о том, какие из имеющихся параметров имеют наибольшее значение для набора баллов по экзамену.
